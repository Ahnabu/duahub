# DuaHub — Islamic Supplications Platform

A multilingual Islamic supplications (Dua) platform built with Next.js 15, featuring duas from the Quran, Sunnah, and the early Muslim scholars (Salaf). Available in **English** and **Bengali (বাংলা)**.

🌐 **Live site:** [duahub.vercel.app](https://duahub.vercel.app)

---

## Features

- **77 authenticated duas** — 24 Quranic, 49 Prophetic (Sunnah), 4 from the Salaf
- **Bilingual** — full English and Bengali for Arabic text, transliteration, meaning, and context
- **Fuzzy search** — fast full-text search powered by Fuse.js across Arabic, transliteration, and meaning
- **Bookmarks** — save duas locally with `localStorage` persistence
- **Share** — native Web Share API with clipboard fallback
- **Dark mode** — system-aware with manual toggle via `next-themes`
- **Category pages** — Quran, Sunnah, Salaf — each with gradient hero, subcategory navigation, and dua sections
- **Ramadan page** — curated Ramadan duas with timeline layout
- **Dua detail pages** — individual pages with transliteration, source, context, tags, and bookmarking
- **SEO** — dynamic `sitemap.xml` and `robots.txt` generated by Next.js App Router APIs
- **Analytics** — Vercel Analytics integration

## Data Structure

All duas live in `src/data/duas.ts` as a typed array. Each dua has:

```typescript
interface Dua {
  id: string;                  // e.g. "q-rep-001", "s-dal-001", "sl-sah-001"
  slug: string;                // URL slug for the dua detail page
  category: "quran" | "sunnah" | "salaf";
  subcategory: DuaSubcategory; // e.g. "repentance", "daily-life", "dhikr"
  subcategory_label_en: string;
  subcategory_label_bn: string;
  purpose_en: string;          // Short purpose / title (English)
  purpose_bn: string;          // Short purpose / title (Bengali)
  source: string;              // e.g. "Qur'an 2:286", "Sahih Bukhari 6306"
  arabic: string;              // Arabic text
  transliteration: string;     // Romanized transliteration
  meaning_en: string;          // English translation
  meaning_bn: string;          // Bengali translation
  tags: string[];              // Search tags
  context_en?: string;         // Optional scholarly context (English)
  context_bn?: string;         // Optional scholarly context (Bengali)
}
```

**ID convention:**
- `q-{subcategory}-{NNN}` — Quranic duas
- `s-{subcategory}-{NNN}` — Sunnah duas
- `sl-{subcategory}-{NNN}` — Salaf duas

## Getting Started

### Prerequisites

- Node.js 18+
- npm / yarn / pnpm

### Install & Run

```bash
# Clone the repository
git clone https://github.com/Ahnabu/duahub.git
cd duahub

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Build for Production

```bash
npm run build
npm run start
```

### Lint

```bash
npm run lint
```

## Dua Categories

### Quran (24 duas)
Direct supplications from the Holy Quran — the highest linguistic and theological standard.

| Subcategory | Examples |
|---|---|
| Repentance | Adam's tawbah (7:23), Yunus in the whale (21:87) |
| Wisdom & Guidance | Musa before Pharaoh (20:25–28) |
| Protection & Success | Surah Al-Falaq, An-Nas, Al-Baqarah duas |
| Family & Progeny | Ibrahim's dua for progeny (14:40–41) |

### Sunnah (49 duas)
Prophetic supplications from authenticated hadith — covering every aspect of daily life.

| Subcategory | Examples |
|---|---|
| Daily Life | Subhanallah wa bihamdihi, 5-part comprehensive dua |
| Repentance | Sayyidul Istighfar (Bukhari 6306) |
| Morning & Evening | Morning dhikr, evening protection |
| Food | Before/after eating, dua for Ramadan sighting |
| Salah | After prayer, istikhara |
| Protection | 8-trait refuge dua (Bukhari/Muslim) |
| Dhikr | Mu'adh's dua after prayer (Abu Dawud 1522) |

### Salaf (4 duas)
Supplications from Sahabah and Tabi'un — early Muslim exemplars.

| Subcategory | Examples |
|---|---|
| Companions (Sahabah) | Abu Bakr's faith dua, Umar's guidance dua |
| Successors (Tabi'un) | Hasan al-Basri's gratitude, Sufyan ath-Thawri's protection |

## Language Support

The language toggle in the header switches between **English** and **Bengali** for:
- Purpose / title of each dua
- Meaning (translation)
- Context and scholarly notes
- Subcategory labels
- UI strings throughout the app

Arabic and transliteration are always shown regardless of language setting.

---

## Adding New Duas

1. Open `src/data/duas.ts`
2. Add a new object to the `duas` array following the `Dua` type
3. Use the ID convention: `q-/s-/sl-` + subcategory abbreviation + 3-digit number
4. Ensure the `subcategory` value exists in `subcategoryMeta` at the bottom of the file
5. Run `npx tsc --noEmit` to verify types before committing
